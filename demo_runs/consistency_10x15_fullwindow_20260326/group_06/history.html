Thermal Optimization History Workspace

这个工作台把初始布局、每轮 LLM 的判断理由、执行动作、验证结果，以及动作在下一轮观察到的真实效果串到同一页里。 顶部三个窗口分别展示初始布局、当前轮仿真、以及下一轮观察到的效果。

Initial Layout

固定展示本组第一轮的布局/温度总览。

Selected Run

当前选中轮次的仿真总览，也就是 LLM 做决策时面对的状态。

Observed Effect

动作的真实效果通常要到下一轮 evaluation 才能观察到。
